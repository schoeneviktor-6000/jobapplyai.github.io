Setup checklist

Before this works, configure Google OAuth for your GitHub Pages domain.

1) Create OAuth Client ID (Web)
2) Add Authorized JavaScript origin: your GitHub Pages domain
3) Add Authorized redirect URI: your signup page URL
4) Publish consent screen and add test users if needed
For a demo, this page stores status locally in your browser (localStorage). For production, use a backend or Firebase for secure token storage.
