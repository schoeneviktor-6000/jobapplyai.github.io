Before someone pays

Questions customers usually have

What counts toward the quota?
Each successful tailored CV generation counts once, whether it starts from the extension or from pasted job text.
Can I upgrade later?
Yes. Users can begin on Free, move into Starter or Plus when needed, and keep Auto-apply as a separate request step.
What if billing is not fully self-serve yet?
The page should still give a clean fallback so interested users can request access instead of hitting a dead end.
Request beta

Auto-apply should be request-only for now

Job fetching and Auto-apply are still market-sensitive. Instead of asking every user to pick a daily application quota, this should now be positioned as a guided beta for users who want more than CV tailoring.

What users should understand

CV Studio is the default product. Auto-apply is the add-on. That keeps the extension globally useful and prevents international users from feeling forced into a Germany-focused workflow.

  • CV Studio works without job fetching
  • Auto-apply depends on sourcing quality and market support
  • The higher-ticket product should begin with a request flow
